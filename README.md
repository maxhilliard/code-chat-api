# crypto-trader-api
A simple cryptocurrency trading bot and a learning exercise in NodeJS

## Prequisites
- Node 7.9.x +
- NPM

## Install Dependencies
`npm install`

## Start Server (local dev only)
`npm start`

## Run Tests
`npm test`
